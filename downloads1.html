Installation media

 The Long Term Support release codenamed 'Andromeda' gives you five years of security patches and updates, it will be supported until August 2030 at least.

  • Q4OS Andromeda, Plasma, live - 64bit / x64 ... 1773 MBytes ... setup guide
  • Q4OS Andromeda, Trinity, live - 64bit / x64 ... 1418 MBytes ... setup guide
  • Q4OS Andromeda, Trinity, install-cd - 64bit / x64 ... 1070 MBytes ... setup guide

 You can download either a lightweight efficient Trinity desktop, or more advanced Plasma desktop edition. Live media allow users to get a quick Q4OS experience, or try it out on a real hardware without installation. If satisfied, an optional installer is available.
 Use the install-cd media for older machines. The minimal hardware requirements:
Plasma desktop - 1GHz CPU / 2GB RAM / 8GB disk
Trinity desktop - 500MHz CPU / 512MB RAM / 6GB disk

Install CD 32bit  [old stable version]

 Use this install-cd for 32bit machines. The old stable release codenamed 'Aquarius' will be supported until June 2028 at least.

Installer for Windows

 The easiest way to put Q4OS alongside Windows. Just run the setup wizard from within Windows and install Q4OS as easily as an application. Windows 11/10/8/7 compatible.
