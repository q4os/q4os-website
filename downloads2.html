Q4OS 6 Andromeda  [testing version]

 Q4OS development version codenamed 'Andromeda' is based on the Debian Trixie testing version, it will become stable in a couple of years.


 The minimal hardware requirements are defined as follows:
Plasma desktop - 1GHz CPU / 1.2GB RAM / 6GB disk
Trinity desktop - 400MHz CPU / 256MB RAM / 3GB disk
 Keep in mind we don't recommend to use testing versions in production environments.
